# 🛒 ShopSphere

ShopSphere is a modern full-stack eCommerce web application built using the MERN Stack.

It allows users to browse products, manage carts, authenticate securely, and place orders through a responsive and clean interface.

---

# 🚀 Features

## 👤 User Features

- User Registration & Login
- Secure Authentication
- Browse Products
- Product Details Page
- Add to Cart
- Remove from Cart
- Checkout Flow

## 🛠️ Admin Features

- Admin Dashboard
- Manage Products
- Manage Orders
- Manage Users

---

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/imapurvsatyam/shopsphere.git
```

## Install Dependencies

```bash
npm install
```

---

# ▶️ Run Project

## Start Backend

```bash
npm run server
```

## Start Frontend

```bash
cd frontend
npm start
```
